## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:jesusvallez/nextjs-ts-eslint-mui.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
3. To start web
   ```sh
   npm run dev
   ```
